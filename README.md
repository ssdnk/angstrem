# ИС УКО АО НПО Ангстрем

## Требования

Установить **Docker Desktop**: https://www.docker.com/products/docker-desktop/

## Запуск

1. Распаковать архив
2. Открыть папку `angstrem` в терминале
3. Выполнить команду:

```
docker compose up --build
```

4. Дождаться появления строки `Application startup complete`
5. Открыть в браузере: **http://localhost**

## Вход

**Сотрудник** — главная страница, выбрать фамилию, имя и подразделение из списка

**Администратор** — нажать «Войти как администратор» внизу страницы

| Логин | Пароль |
|---|---|
| admin | admin123 |
| hr_specialist | angstrem2024 |

## Остановка

```
docker compose down
```

База данных сохраняется между запусками.
